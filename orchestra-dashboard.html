How to Refresh This Dashboard

This is a static HTML file with data baked in. To update it with current Orchestra state, use one of these methods:

Option 1: From any Claude Code session, run:

/orchestra refresh-dashboard

Option 2: From the parent workspace session, ask Claude to regenerate the dashboard.

The /orchestra refresh-dashboard command reads all task and inbox JSON files and rewrites the ORCHESTRA_DATA block in this HTML file.
